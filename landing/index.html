So funktioniert's

In 4 Schritten zum Meeting-Protokoll

Kein Setup, kein Account, keine Cloud. Einfach starten.

Aufnahme starten

Klicke im Dashboard auf "Aufnahme starten". Minuta erfasst Mikrofon und System-Audio automatisch.

Live transkribieren

Sieh das Transkript in Echtzeit im Browser. Sprecher werden automatisch unterschieden (Ich vs. Teilnehmer).

Zusammenfassen

Nach dem Meeting generiert GPT-4o oder ein lokales LLM eine strukturierte Zusammenfassung mit Action Items.

Exportieren

Per Webhook versenden. Automatisch in Notion ablegen, weiterverarbeiten oder per E-Mail verteilen.

Tech Stack

Gebaut für Performance

Optimiert für Apple Silicon mit modernsten Technologien.

Audio Capture

Swift CLI mit ScreenCaptureKit und AVAudioEngine. Native macOS Integration ohne Drittanbieter-Treiber.

MLX Whisper

Apples MLX Framework für lokale Whisper-Inferenz. Nutzt Neural Engine und Metal GPU auf M1/M2/M3/M4.

FastAPI Backend

Python Backend mit async SQLite, WebSocket Broadcasting und konfigurierbaren LLM-Providern.

Next.js 16 Frontend

React 19 mit shadcn/ui, Tailwind CSS v4, Lucide Icons. Moro Vision Green Theme.
